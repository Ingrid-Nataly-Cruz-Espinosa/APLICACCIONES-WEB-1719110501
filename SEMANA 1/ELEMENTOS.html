Elementos de raiz basicos en una pagina web


!DOCTYPE HTML

nos sirve para indicar que nuestro documento está escrito siguiendo la estructura determinada por un DTD concreto.

HTML

Da a conocer la raíz de un documento HTML o XHTMl y asi de la misma manera los elementos deben ser descendientes de este elemento

HEAD

Representa los metadatosdel documento asi como lo so enlaces, definiciones de scripts y las hojas de estilo.

TITLE

Como su nombre lo dice hace refrencia al título del documento, y es el que se muestra en los navegador y pestañas al momento de abrir alguna paginas solo acepta texto

BASE

Da a conocer las URL es la encragada de las URLS en los sitios web

LINK

Se encarga de hacer una conexion entre JavaScript y CSS externos con el documento mas reciente HTML

META

Define los metadatos que no pueden ser definidos usando otro elemento HTML.

STYLE

Etiqueta de estilo que se utliza para escribir CSS en línea.

P

Define un parrafo.

Secciones en HTML

BODY

Representa el contenido principal de un documento HTML.

SECTION

Sección en un documento.

NAV

Da a conocersección que solamente cuenta con enlaces de navegación

ARTICLE

Define contenido autónomo que podría existir independientemente del resto del contenido.

ASIDE

Define algunos contenidos vagamente relacionados con el resto del contenido de la página. Si es removido, el contenido restante seguirá teniendo sentido

ADDRESS

Sección que define, que contiene información de contacto.
